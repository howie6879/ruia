    Installation

    Ruia is based on Python 3.6+. Check your python version before continue.

    python --version

    Installation

    # For Linux & Mac
pip install -U ruia[uvloop]

# For Windows
pip install -U ruia

# Install from github for new features
pip install git+https://github.com/howie6879/ruia
