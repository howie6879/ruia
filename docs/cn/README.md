# Ruia

**Ruia**是一个基于`asyncio`和`aiohttp`的异步爬虫框架，它的目标是让你更加方便且迅速地编写出属于自己的爬虫

很高兴你能使用**Ruia**来实现爬虫程序，不过在编码之前，希望你能通读此文档，因为它包含了**Ruia**的使用方法以及一些基础概念介绍
