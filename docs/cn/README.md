## aspider 中文文档

**aspider**是一个基于`asyncio`和`aiohttp`的异步爬虫框架，它的目标是让你更加方便且迅速地编写出属于自己的爬虫

很高兴你使用**aspider**来实现你想要的爬虫，不过在编码之前，希望你能通读此文档，因为它包含了**aspider**的使用方法以及一些基础概念介绍
